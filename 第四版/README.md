# HealthGame - 健康生存模拟器

欢迎来到 **HealthGame（健康生存模拟器）**！这是一款主打【硬核与真实】的模拟生存类独立游戏。

## 🎮 游戏简介

在这个游戏中，你最大的敌人是你自己的【身体】和【时间】。
你需要从 0 岁开始，一直努力活到人类极限的 110 岁。在此期间，你需要时刻监控自己的血压、血脂、血糖和免疫力。稍有不慎，一个小小的炎症就可能引发蝴蝶效应，最终演变成不可逆的全身性衰竭。

## ✨ 核心特色

- 🫀 **真实残酷的生理系统**：内置五大核心器官（心、肺、肝、胃、皮肤）和 50 多种极其硬核的现实疾病。
- 🎲 **天赋与命运的博弈**：自由分配【家境】、【气运】、【健康】、【科研】四大天赋，不同加点对应完全不同的生存策略。
- 🧬 **基因大限与续命机制**：在 80 岁和 100 岁面临人类基因极限时，利用顶级的科研能力研发特效药和长寿技术，支付高昂的续命费用打破极限！
- 🩺 **深度诊疗体系**：涵盖药剂、手术、靶向药、免疫抑制剂、体外生命支持（ECMO）等多种治疗手段，针对晚期绝症甚至需要分阶段靶向治疗。

## 🚀 如何运行与编译

本项目基于 **C++ 17** 和 **Qt 6** 框架开发。

### 直接游玩 (Windows)
你可以直接在 [Releases](../../releases) 页面下载最新打包好的绿色免安装版，解压后双击 `HealthGame.exe` 即可游玩。

### 源码编译
如果你想自行编译或修改游戏内容：
1. 安装 **Qt 6** 及 **MinGW 64-bit** 编译工具链。
2. 安装 **CMake**。
3. 在 Qt Creator 中打开本项目的 `CMakeLists.txt`。
4. 配置并构建项目（推荐使用 Release 模式）。
5. 运行！

## 🛠️ 如何添加自己的疾病或药物？

游戏的疾病和药物数据核心位于 `Database.cpp`。你可以参考其中的 `ADD_DISEASE` 和 `ADD_MEDICINE` 宏定义，非常方便地扩展你想要的变态疾病或神奇药物！

```cpp
// 示例：添加一种新的疾病
ADD_DISEASE("D_NEW", "赛博精神病", "大脑", 18, 100, 0.05, 5, "幻听，暴躁，记忆错乱", 30, "【诊疗指南】义体过度改造导致。需使用【神经稳定剂】。");
```
