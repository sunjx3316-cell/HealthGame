# 【南开大学26C++】器官保卫战 (HealthGame)

## 1. 项目问题描述
本项目旨在开发一款基于 C++ 和 Qt 框架的生存策略模拟游戏——《器官保卫战》。玩家需要管理角色的健康指标（血压、血糖、血脂）、免疫力、金钱，以及分配初始天赋，在生老病死中尽可能长久地生存。

## 2. 项目实际意义
1. **健康教育科普**：模拟高压科研对身体指标的负面影响，呼吁关注健康，定期体检。
2. **编程能力实践**：综合运用 C++ 面向对象编程、Qt 信号与槽机制、状态机设计模式。

## 3. 技术路线及算法描述
1. **技术路线**：使用 C++17 标准，基于 Qt Widgets 框架开发纯 2D 图形界面。UI 通过 QTabWidget 划分为独立模块。
2. **核心算法**：
   - 时间驱动引擎：使用 QTimer 作为游戏心跳驱动年龄和指标变化。
   - 动态概率模型：采用 QRandomGenerator 结合玩家天赋动态修正疾病发生率。
   - 隐性疾病状态机：针对 5 级绝症，设计了“潜伏期 -> 中期 -> 晚期爆发”的状态机算法。

## 4. 开发过程中遇到的难点及解决方法
1. **难点1：UI 联动与精准警报**。解决：引入 HTML 富文本格式动态渲染 QLabel 进行独立着色。
2. **难点2：后期数值膨胀**。解决：重构经济系统，将收入与科研天赋绑定，引入高昂的特效药和多阶段治疗费用。
3. **难点3：绝症的真实感**。解决：独创隐性疾病进展算法，逼迫玩家养成每 3 年体检的习惯。
